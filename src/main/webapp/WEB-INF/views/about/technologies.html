About People Books

What is this "Spring Library Application" about?

The local library wants to switch to digital book counting, I have implemented a web application for them. Librarians have the opportunity to register readers, give them books and release the books after the reader returns book back to the library.

To start using the app, enter books in the "Books" tab and enter people in the "People" tab

Functionality:

  • pages for adding, changing and deleting a person
  • pages for adding, editing, and deleting a book
  • page with a list of all people (people are clickable - when clicked, go to person's page)
  • page with a list of all books (books are clickable - when clicked, go to book page
  • a person's page showing the values of his fields and a list of the books he has taken. If the person hasn't picked up any books, the text "Person haven't picked up any books yet.
  • Page of a book showing the values of the margins of that book and the person's name, who took this book. If this book has not been borrowed by anyone, the text "This the book is free"
  • On the book page, if the book is taken by a person, a button appears next to his name "Release book". This button is pressed by the librarian when the reader returns this book back to the library. After pressing this button the book again becomes loose and disappears from a person's book list
  • On the book page, if the book is free, a drop-down list appears with all people and the "Assign Book" button. This button is pressed by the librarian when the reader wants to take this book home. After pressing this button, the book starts to belong to the selected person and appears in his list books
  • All input fields are validated - using @Valid and Spring Validator
